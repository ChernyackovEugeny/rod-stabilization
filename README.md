# rod-stabilization

Задача стабилизации перевернутого стержня



Необходимые библиотеки: matplotlib.pyplot, matplotlib.animation, numpy, multiprocessing, os, pickle



в main.py пример использования и визуализации одного из агентов(чтобы видеть процесс обучения(выводимые rewards), дстаточно раскомментировать одну строку с "print" в соответствующем агенте)

в public полученные графики, последняя версия отчета



src/ACagent.py, src/BRSagent.py, src/HCagent.py - классы агентов

src/Model.py - класс окружающей среды

src/Visualize.py - класс-визуализатор

src/validation.py - подбор гиперпараметров(для использования раскомментировать нужный кусок, одновременно может работать только один из трех кусков)



для получения данных и графиков:
запустить data\_collect.py, затем results.py

графики сохранятся в папку src/results
